# BHELA Blog — Implementation Plan

> Project-based plan. All future plans live in `wp-content/docs/plans/` (committed with the repo), not in any global/user directory.

## Context

The site converts well but has no content marketing surface. A Bangla-first travel blog ("হাওর জার্নাল") targets organic search for Tanguar Haor trip queries (best season, what to pack, family safety, full-moon trips), feeds the booking funnel, and reuses infrastructure the theme already has: `index.php` renders a `.posts-grid` of `.post-card`s, `single.php` renders a post with `.page-hero` + `.entry-content`, and CSS for both exists in `style.css` (§ posts-grid/post-card). What's missing: a blog page + nav link, category/tag surfaces on cards and posts, post navigation, related posts, SEO schema, and a booking CTA on posts. No comments (spam burden for a small operator; WhatsApp CTA replaces them).

## Decisions

- Language: Bangla-first, English keywords inline (matches rest of site).
- Comments: **off** (`add_filter('comments_open','__return_false')` on posts) — CTA banner instead.
- Blog URL: `/blog/` via a "ব্লগ" page assigned as `page_for_posts`.
- Categories seeded: ভ্রমণ গাইড (travel-guide), হাওরের খবর (haor-news), টিপস (tips).

## Changes (theme `bhela`, no plugin work)

1. **functions.php**
   - `bhela_auto_setup()`: create page `blog` (title "ব্লগ"), set `page_for_posts`; append "ব্লগ" to the BHELA Primary menu after FAQ. Seed the 3 categories. Idempotent like existing setup (get_page_by_path guard).
   - `bhela_fallback_menu()`: add ব্লগ item.
   - New helper `bhela_reading_time( $post_id )` — word count / 200, returns "X মিনিট পড়া".
   - Article JSON-LD on `is_single()` (mirror `bhela_schema()` pattern: headline, datePublished/Modified, image from thumbnail, publisher BHELA).

2. **index.php** (blog archive) — upgrade in place:
   - `.page-hero` title: "হাওর জার্নাল" + lead; on category/tag views keep `get_the_archive_title()`.
   - `.post-card`: add category chip (reuse `.tag` pill styles), date + reading time meta line, `reveal` animation class.
   - Keep `the_posts_pagination()`; style `.page-numbers` in style.css (currently unstyled — orange current, ink hover).

3. **single.php** — upgrade in place:
   - Hero: category chip + date + reading time under title.
   - After content: tags row (`.tag` pills), **prev/next post nav** (two cards, `previous_post_link`/`next_post_link` markup), **related posts** (3 `.post-card`s, same category, `orderby => rand`, exclude current), then the existing `.cta-banner` pattern ("হাওর নিজে দেখতে চান?" → বুক করুন + WhatsApp, reuse `bhela_wa_link()`/`bhela_page_url('book-now')`).
   - `comments_open` filter keeps comment UI away.

4. **style.css** — small additions only (reuse existing tokens/classes):
   - `.post-card__meta` (date · reading time), `.post-nav` 2-col grid (stacks ≤640), `.related-posts` heading spacing, `.page-numbers` pagination pills. Bump version.

5. **Sample content**: 3 seed posts (draft) so the grid isn't empty at launch — titles from the FAQ's strongest queries (সেরা সময়, কী কী নেবেন, ফ্যামিলি নিরাপত্তা).

## Reuse (do not rebuild)

- `.posts-grid`, `.post-card`, `.page-hero`, `.entry-content`, `.tag--*`, `.cta-banner`, `.reveal` — all in `themes/bhela/style.css`.
- `bhela_wa_link()`, `bhela_page_url()`, `bhela_money()` in `themes/bhela/functions.php`.
- Schema pattern: `bhela_schema()` (functions.php) and FAQ JSON-LD (template-faq.php).

## Milestones

- **M1** Blog page + menu + categories (auto-setup) → `/blog/` renders existing grid.
- **M2** index.php + pagination styling + card meta.
- **M3** single.php: meta, tags, post nav, related, CTA, Article schema, comments off.
- **M4** Seed posts, mobile pass (grid 1-col, nav cards stack), graphify update.

## Verification

1. CLI (Local PHP + mysqli port 10028): run auto-setup, assert `get_option('page_for_posts')` set, menu has ব্লগ, categories exist.
2. `curl /blog/` → 200, `.posts-grid`, pagination present with >10 posts (temporarily set posts_per_page low to test).
3. Publish a post with thumbnail + category + tags → `curl` the permalink: hero meta, tags row, post-nav, related, CTA banner, `"@type":"Article"` JSON-LD, **no** comment form.
4. Browser (Chrome tool): visual check desktop; forced-width layout probe ≤640 (cards stack, nav stacks); console clean.
5. Regression: homepage + 7 pages still 200 with markers; theme version bumped for cache-bust.
